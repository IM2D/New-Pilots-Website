-- Local Python Environment setup ---
-- (Assume you have PYTHON installed, check https://wiki.python.org/moin/BeginnersGuide/Download if not)--
-- Run below steps in command line window
1) cd YOUR_PYTHON_PROJECT folder(contains github source code)
2) run following commend to setup virtial environment
  python -m venv dev
3) run the following command to active virtual environment
  dev\Scripts\activate.bat
4) run the following command to intall flask package
  pip install flask flask-sqlalchemy flask-login
5)Ensure Python interpreter is configured
6) run the following command to create a folder store all uploads
  mkdir c:\uploads

-- in IDE(Visual Code or PyCharm IDE)
run __init__.py
